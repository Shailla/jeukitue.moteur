* Installer MinGW 64
 - Télécharger "msys2-x86_64-20160205.exe" ici "http://msys2.github.io/"
 - Ouvrir la console MSYS (la suite se fait dedans)
 - Tout mettre à jour (procédure prise ici "http://msys2.github.io/") :
   - pacman -Sy pacman
   - pacman -Syuu (plusieurs fois si nécessaire)
 - Mettre à jour les cibles des raccourcis Windows dans le menu Démarrer :
   - MinGW-w64 Win32 Shell : C:\msys64\msys2_shell.cmd -mingw32
   - MinGW-w64 Win64 Shell : C:\msys64\msys2_shell.cmd -mingw64
   - MSYS2 Shell : C:\msys64\msys2_shell.cmd -msys
 - Installer les utilitaires "pacman -S perl tar make"
 - Installer la toolchain 64 bits "pacman -S mingw-w64-x86_64-toolchain"
 - Installer FreeType en lançant "pacman -S mingw-w64-x86_64-freetype"
 - Installer SDL (1.2!) en lançant "pacman -S mingw-w64-x86_64-SDL"
 - Installer SDL_net en lançant "pacman -S mingw-w64-x86_64-SDL_net"
 - Installer SDL_image en lançant "pacman -S mingw-w64-x86_64-SDL_image"
 - Installer libpng pour Agar en lançant "pacman -S mingw-w64-x86_64-libpng"
 - Lancer CMake depuis la console Mingw64 (/c/Program Files (x86)/cmake/bin, puis ./cmake-gui.exe)
 
 Mettre C:\msys64\mingw64\bin dans le path
 
 
* Installer la lib SDL (version 1.2.15) ==> Inutile 
 - Télécharger la lib SDL "SDL-devel-1.2.15-mingw32.tar.gz" deuis ici "https://www.libsdl.org/download-1.2.php"
 - Décompresser la lib dans un répertoire quelconque
 - Dans la console MSYS, se mettre dans ce répertoire
 - Lancer "./configure"

 - Lancer "make install"

* Installer la lib Agar
 - Télécharger la lib Agar "agar-1.5.0.tar.gz" depuis ici "http://libagar.org/download.html"
 - Décompresser la lib dans un répertoire quelconque
 - Dans la console Mingw64, se mettre dans ce répertoire
 - Lancer "./configure --enable-debug --with-{freetype,sdl,gl}"

 - Lancer "make install"
